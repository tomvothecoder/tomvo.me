# tomvo.me

Tom Vo's personal website, built with React, TypeScript, and Vite.

## Available Scripts

In the project directory, you can run:

### `pnpm dev`

Starts the dev server. Open [http://localhost:5173](http://localhost:5173) to view the app. The page will reload when you edit files.

### `pnpm build`

Builds the app for production into the `build` folder. The output is minified and ready to deploy.

### `pnpm preview`

Serves the production build locally so you can test it before deploying.

### `pnpm test`

Runs the test suite with Vitest in watch mode.
